```
[ユーザー操作]
   ↓
[フロントエンド] → API通信 → [バックエンド(Express)]
   ↓                                  ↓
   ↓                             DBアクセス
   ↓                                  ↓
[ブラウザに結果が返る] ←──────────────


┌─────────────────────────────────────────────┐
│      フロントエンド（React / Next.js）　　　　│
│---------------------------------------------│
│ ・ユーザーが操作する画面                      │
│ ・fetch() や axios で API にリクエスト送信    │
└─────────────────────────────────────────────┘
                 │
                 ▼
        HTTPリクエスト（例: GET /api/users）
                 │
                 ▼
┌────────────────────────────────────────────┐
│      バックエンド（Next.js / Express）       │
│--------------------------------------------│
│ ・APIルート（/api/...）がリクエストを受け取る │
│ ・Expressがルーティング処理を行う            │
│ ・Morgan がアクセスログを記録 📝            │
│ ・ビジネスロジックでDBへ問い合わせ            │
└────────────────────────────────────────────┘
                 │
                 ▼
┌──────────────────────────────────────────┐
│           データベース（DB）              │
│------------------------------------------│
│ ・ユーザー情報や投稿データなどを保存        │
│ ・MySQL / PostgreSQL / MongoDB など       │
└──────────────────────────────────────────┘
                 ▲
                 │
        クエリ結果（データを返す）
                 │
                 ▼
┌──────────────────────────────────────────┐
│      バックエンド（Next.js / Express）     │
│------------------------------------------│
│ ・DBから受け取ったデータを整形             │
│ ・レスポンスとしてJSONを返す               │
│ ・Winston がアプリ全体のログを記録 🪵      │
└──────────────────────────────────────────┘
                 │
                 ▼
       HTTPレスポンス（JSON形式のデータ）
                 │
                 ▼
┌──────────────────────────────────────────┐
│      フロントエンド（React / Next.js）     │
│------------------------------------------│
│ ・受け取ったJSONデータを画面に表示           │
│ ・例: ユーザー一覧や投稿内容の描画           │
└──────────────────────────────────────────┘
